在线考试系统

项目仓库位置

项目简介

这是一个基于Spring Boot开发的在线考试系统，支持多角色用户（管理员、教师、学生）进行在线考试管理。系统提供了完整的考试流程管理，包括试题管理、试卷生成、考试发布、在线答题和成绩统计等功能。

本项目基于网络上的开源项目苍穹外卖，利用其思想和代码基础进行的开发，借鉴了其优秀的架构设计。

功能特点

多角色权限管理

  • 系统管理员：管理全局试卷库和用户权限
  • 教师：创建和管理试卷、题库，发布考试
  • 学生：参加考试、查看成绩

核心功能模块

1. 用户认证系统

  • JWT Token身份验证
  • 多角色登录（管理员、教师、学生）
  • 密码加密存储

2. 题库管理

  • 支持多种题型管理
  • 按科目分类管理题目
  • 教师可添加和维护题目

3. 试卷管理

  • 自动生成试卷功能
  • 手动组卷功能
  • 试卷状态管理（草稿、发布等）

4. 考试管理

  • 考试发布和权限控制
  • 考试时间限制设置
  • 按专业和班级发布考试

5. 在线考试

  • 友好的答题界面
  • 实时倒计时
  • 答题卡导航
  • 自动保存答题进度

6. 成绩管理

  • 自动评分系统
  • 成绩排行榜
  • 多维度成绩统计

技术架构

后端技术栈

  • Spring Boot 3.5.6：核心框架
  • MyBatis：数据库访问层
  • JWT：用户身份认证
  • Druid：数据库连接池
  • Lombok：简化Java代码
  • Knife4j：API文档管理

前端技术栈

  • Thymeleaf：服务端模板引擎
  • HTML/CSS/JavaScript：基础前端技术
  • Axios：HTTP客户端
  • 响应式设计，适配移动端

数据库

  • MySQL：关系型数据库

系统要求

  • JDK 21+
  • Maven 3.6+
  • MySQL 8.0+

快速开始

1. 环境准备

确保已安装JDK 21、Maven和MySQL

2. 数据库配置

  1. 创建数据库：exam_data
  2. 在 application-dev.yml 中配置数据库连接信息

3. 项目构建

4. 访问系统

项目启动后，访问 http://localhost:8080 进入登录页面

项目结构

主要页面

  1. 登录页面 (/) - 用户登录入口
  2. 仪表板 (/dashboard) - 用户主界面，根据角色显示不同功能
  3. 管理员页面 (/admin) - 管理员功能界面
  4. 教师页面 (/teacher) - 教师功能界面
  5. 学生页面 (/student) - 学生功能界面
  6. 考试页面 (/exam/{examId}) - 在线考试界面

开发团队

本项目由个人开发完成。

许可证

本项目仅供学习交流使用。
